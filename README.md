# Pomodoro Timer (Basic)

Simple Pomodoro timer built with HTML and JavaScript.

## Features
- Start / Pause / Reset
- Work mode (25 minutes)
- Break mode (5 minutes)
- Time displayed as mm:ss

## How to run
Just open the `index.html` file in your browser.
